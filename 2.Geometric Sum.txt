Problem Level: MEDIUM
Problem Description:
Given k, find the geometric sum i.e.
1 + 1/2 + 1/4 + 1/8 + ... + 1/(2^k)

using recursion.
Input format :
Integer k

Output format :
Geometric sum

Constraints :
0 <= k <= 1000

Sample Input 1 :
3

Sample Output 1 :
1.875

Sample Input 2 :
4

Sample Output 2 :
1.93750


Code

using namespace std;
double geoSum(int a,int b)
{
    if(a==b)
    {
        return (double)1/pow(2,a);

     return ((double)(1/pow(2,b)+geoSum(a,b+1)));
}
